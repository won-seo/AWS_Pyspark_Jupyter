1. Create AWS account

 

2. S3: create bucket and upload data

 

 

3. EC2: Create keypair

NOTE: you may need to change the access permission of the keypair

 

 

4. EC2: Create security group

SSH: 22 | 0.0.0.0/0

TCP: 8888 | 0.0.0.0/0

 

 

5. EMR: Create cluster

Select the features you need

Pick the Instance type and number of instance

You can add bootstrap to download any applications in cluster

 

Make sure to pick the keypair and security group you made in previous step

Wait until the cluster is ‘Waiting’

 

 

6. Terminal: install anaconda3 in cluster and set spark to use jupyter

 

 

7. Terminal: ssh -I ~/keypair.pem hadoop@public DNS

NOTE: you can find your public DNS when you click the cluster in EMR

 

 

8. Terminal: pyspark

Copy the token

 

 

9. Browser: localhost:8888

 

 

 

 

 

10. Use the copied token to login and enjoy using AWS pyspark jupyternotebook
